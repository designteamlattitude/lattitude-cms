# catalog-site-base

Base Strapi CMS for catalog-driven websites.

This repository contains the Strapi Cloud setup used to manage content for catalog-based sites, including:
- Global layout content (navigation, footer, branding)
- Product catalogs (PLP / PDP)
- Brands
- Pages such as Home, About, Contact
- Reusable content structures intended for replication across similar projects

The CMS is designed to be:
- Reusable across multiple websites
- Easy to duplicate and customize per project
- Decoupled from frontend implementations (Astro or others)

This repository serves as a CMS base and is intended to be duplicated for each catalog site.
